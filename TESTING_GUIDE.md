cd c:\c_projects\flixa-main

http://localhost:8083

## ✅ قائمة الاختبار الشاملة

### 📱 الصفحات المطلوب اختبارها

#### 1. الصفحة الرئيسية

- **المسار**: `http://localhost:8083/`
- **ما يجب التحقق منه**:
  - ✅ عرض البطاقات الإحصائية الـ 4
  - ✅ عرض الرسوم البيانية
  - ✅ عرض الروابط السريعة
  - ✅ البيانات تُحدّث بشكل ديناميكي

#### 2. إدارة المستخدمين

- **المسار**: `http://localhost:8083/users`
- **ما يجب التحقق منه**:
  - ✅ عرض قائمة المستخدمين (≥ 25 مستخدم)
  - ✅ البحث يعمل بشكل صحيح
  - ✅ يمكن تغيير حالة المستخدم
  - ✅ عرض المعلومات (الهاتف، الطلبات، الإنفاق)
  - ✅ الألوان تتغير حسب الحالة

#### 3. إدارة العمولات

- **المسار**: `http://localhost:8083/commissions`
- **ما يجب التحقق منه**:
  - ✅ عرض قائمة العمولات
  - ✅ عرض نوع العمولة (نسبة/مبلغ)
  - ✅ عرض الحالة (نشط/غير نشط)
  - ✅ البحث يعمل

#### 4. التقارير والإحصائيات

- **المسار**: `http://localhost:8083/reports`
- **ما يجب التحقق منه**:
  - ✅ عرض 4 بطاقات ملخص
  - ✅ عرض جدول التقارير
  - ✅ عرض حالات التقارير

#### 5. التقارير المالية

- **المسار**: `http://localhost:8083/financial-reports`
- **ما يجب التحقق منه**:
  - ✅ عرض 4 بطاقات مالية
  - ✅ عرض الإيرادات والأرباح
  - ✅ عرض جدول التقارير المالية

#### 6. إدارة المحتوى

- **المسار**: `http://localhost:8083/content`
- **ما يجب التحقق منه**:
  - ✅ عرض قائمة الصفحات (6 صفحات)
  - ✅ عرض اللغة (عربي/إنجليزي)
  - ✅ عرض حالة الصفحة

#### 7. إدارة الرسائل

- **المسار**: `http://localhost:8083/messages`
- **ما يجب التحقق منه**:
  - ✅ عرض قائمة الرسائل
  - ✅ عرض عدد الرسائل الجديدة
  - ✅ تحديد الأولوية بالألوان
  - ✅ البحث يعمل

#### 8. إدارة الإشعارات

- **المسار**: `http://localhost:8083/notifications`
- **ما يجب التحقق منه**:
  - ✅ عرض قائمة الإشعارات
  - ✅ عرض بطاقات الملخص (4 بطاقات)
  - ✅ يمكن إنشاء إشعار جديد (زر "إشعار جديد")
  - ✅ يمكن فتح Dialog إنشاء الإشعار

## 🔍 اختبارات الميزات

### 🔎 البحث والتصفية

```bash
# في صفحة المستخدمين/الرسائل
- ابحث عن "محمد"
- ✅ يجب أن تظهر النتائج المطابقة
- ✅ البحث يجب أن يعمل على الاسم والبريد
```

### 🎨 الألوان والحالات

```bash
# تحقق من الألوان:
- أخضر: نشط/مكتمل ✅
- برتقالي: تحذير ⚠️
- أحمر: خطأ ❌
- رمادي: افتراضي
```

### 📱 الاستجابة (Responsive)

```bash
# اختبر على أحجام مختلفة:
- سطح المكتب (1920px) ✅

- الهاتف (375px) ✅
```

### 🌍 الدعم متعدد اللغات

```bash
# تحقق من:
- ✅ النصوص بالعربية

```

## 🐛 اختبارات الأخطاء الشائعة

### ❌ إذا لم تحمل الصفحة:

```
الحل:
1. تحقق من console (F12)
2. تأكد من تشغيل pnpm dev
3. حاول إعادة التحديث (Ctrl+R)
4. امسح cache: Ctrl+Shift+Delete
```

### ❌ إذا لم تظهر البيانات:

```
الحل:
1. تحقق من import البيانات
2. استخدم console.log للتصحيح
3. تأكد من وجود المكون في الصفحة
```

### ❌ إذا لم تعمل الأيقونات:

```
الحل:
1. تحقق من ICONS.navbar في config-icons.tsx
2. استخدم أيقونة موجودة
3. تحقق من المسار: /assets/icons/
```
