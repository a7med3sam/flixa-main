# 📊 Admin Dashboard - مشروع لوحة التحكم الإدارية

## نظرة عامة

مشروع لوحة تحكم إدارية كاملة لتطبيق موبايل، مبني بـ **Next.js 15** و **Material-UI 6**، يوفر جميع أدوات الإدارة اللازمة.

## 🎯 الأقسام الرئيسية

### 1️⃣ **إدارة المستخدمين**

- عرض قائمة المستخدمين
- تفعيل/تعطيل حسابات المستخدمين
- عرض معلومات المستخدم الشاملة
- تحديث حالة المستخدم (نشط، غير نشط، معلق، محظور)
- 📍 المسار: `/users`

### 2️⃣ **إدارة العمولات**

- عرض قائمة العمولات على المدفوعات
- تحديث نسب/مبالغ العمولات
- تفعيل/تعطيل العمولات
- عرض إحصائيات العمولات
- 📍 المسار: `/commissions`

### 3️⃣ **التقارير والإحصائيات**

- عرض إحصائيات المستخدمين
- تقارير العمليات والمعاملات
- إحصائيات الطلبات
- تقارير الإيرادات
- 📍 المسار: `/reports`

### 4️⃣ **التقارير المالية**

- عرض الإيرادات والمصروفات
- حساب الأرباح والخسائر
- تقارير الدفع حسب الطريقة
- تحليل الإنفاق
- 📍 المسار: `/financial-reports`

### 5️⃣ **إدارة المحتوى**

- تحرير صفحة "عن التطبيق"
- تحرير الشروط والأحكام
- تحرير سياسة الخصوصية
- دعم لغات متعددة (عربي/إنجليزي)
- 📍 المسار: `/content/*`

### 6️⃣ **إدارة الرسائل**

- عرض الرسائل الواردة من المستخدمين
- تصنيف الرسائل حسب الأولوية
- الرد على الرسائل
- 📍 المسار: `/messages`

### 7️⃣ **إدارة الإشعارات**

- إنشاء وإرسال إشعارات جديدة
- دعم إشعارات متعددة اللغات (عربي/إنجليزي)
- اختيار المستقبلين (الكل، نشطين، غير نشطين)
- تتبع إحصائيات الإرسال
- 📍 المسار: `/notifications`

## 🚀 كيفية البدء

```bash
# تثبيت المتطلبات
pnpm install

# تشغيل بيئة التطوير
pnpm dev

# بناء المشروع
pnpm build

# بدء السرفر
pnpm start
```

## 📱 تقنيات المستخدمة

- **Next.js 15**: Framework React الحديث
- **Material-UI 6**: مكتبة المكونات الحديثة
- **TypeScript**: لضمان أمان الكود
- **next-intl**: دعم متعدد اللغات
- **React Hook Form**: إدارة النماذج
- **Recharts**: الرسوم البيانية (اختياري)
